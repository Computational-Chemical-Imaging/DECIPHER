Both Matlab and Python versions for background removal via arPLS are included

Under the corresponding folder, open the main script main_MIP_arPLS.m or main_MIP_arPLS.py and run the script section by section.

For demonstration, use the MIP image provided in the folder MIP_arpls. Filename is G2_MIP_DN_stab.tif. Data can be downloaded at https://doi.org/10.6084/m9.figshare.31014862

Store the target image data in the specified folder. The output data will be saved under the same folder.

The code is adapted from https://github.com/CCampJr/CRIkit/
