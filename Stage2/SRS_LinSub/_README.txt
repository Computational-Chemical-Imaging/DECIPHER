Both Matlab and Python versions for background extraction through linear model are included

Under the corresponding folder, open the main script main_SRS_LinSub.m or main_SRS_LinSub.py and run the script section by section.

For demonstration, use Dataset 1's Stage 1 output image. Filename is U87_CH_SRS_raw_drift_crtd_denoised.txt. Data can be downloaded at https://doi.org/10.6084/m9.figshare.31014862

Store the target image data in the specified folder. The output data will be saved under the same folder.
