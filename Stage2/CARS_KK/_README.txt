Both Matlab and Python versions for KK phase retrieval are included.

Under the corresponding folder, open the main script main_KK.m or main_KK.py and run the script section by section.

For demonstration, use Dataset 2's Stage 1 output image. Filename is OVCAR5_CH_CARS_raw_shading_crtd_SPEND.tif. Data can be downloaded at https://doi.org/10.6084/m9.figshare.31014862

Store the target image data in the Data folder. The output data will be saved under the same folder.

The code is adapted from https://github.com/CCampJr/CRIkit.
