Both Matlab and Python scripts for non-negative pixel-wise LASSO unmixing are provided.

Under the corresponding folder, open the main script main_LASSO.m or main_LASSO.py and run the script section by section. This script is intended to process one image only.

For demonstration, use Dataset 1's Stage 1 output image. Filename is U87_CH_SRS_raw_drift_crtd_denoised.txt. Data can be downloaded at https://doi.org/10.6084/m9.figshare.31014862

For batch processing, we provide main_LASSO_Batch.m and main_LASSO_Batch.py. The script is written to process Dataset 4: Fingerprint SRS imaging of C. elegans. Data can be downloaded at https://doi.org/10.6084/m9.figshare.31014862
