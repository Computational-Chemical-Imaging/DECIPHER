For MATLAB demo, add folder "bm4d_matlab_package_4.2.5" to path, store the target image in the "Data" directory, and run the MATLAB script "main_hs_denoising.m". 

Use "U87_CH_SRS_raw_drift_crtd.txt" in Dataset 1 for demonstration. Put the image file in the same directory std is set to 0.05 in the demonstration. Output is named "U87_CH_SRS_raw_drift_crtd_denoised.txt" and is saved in the same data directory.


For Python demo, first install the software package using the command "pip install bm4d". Alternatively, the package can be downloaded from the webpage "https://webpages.tuni.fi/foi/GCF-BM3D/index.html#ref_software". Store the target image in the 'Data' directory, and run the MATLAB script "main_hs_denoising.py". 

Use 'U87_CH_SRS_raw_drift_crtd.txt' in Dataset 1 for demonstration. Put the image file in the same directory std is set to 0.05 in the demonstration. Output is named 'U87_CH_SRS_raw_drift_crtd_denoised.txt' and is saved in the same data directory.
